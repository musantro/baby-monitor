Baby Monitor - Local Help

This application is designed to run within your local network.

Quick Start

  1. Open this app from the same server URL on all devices in your LAN.
  2. On one device, select Use as Baby Device and start camera.
  3. On parent devices, select Use as Parent Device and request connection.

Security Notes

  • Audio/video stream is peer-to-peer (WebRTC).
  • Server only handles signaling via /api/v1/exchange.
  • Deploy behind your local firewall to avoid WAN exposure.
